### рҹ”¬ &nbsp;`[ 01 ]` &nbsp;Wafer Defect Detection & Classification

![Status](https://img.shields.io/badge/Status-рҹҹў%20Active-00d4ff?style=flat-square&labelColor=0d1117)
![Type](https://img.shields.io/badge/Type-AI%20%2F%20Computer%20Vision-7b2ff7?style=flat-square&labelColor=0d1117)
![Stack](https://img.shields.io/badge/Stack-PyTorch%20В·%20Grad--CAM%20В·%20Streamlit-FF4B4B?style=flat-square&labelColor=0d1117)

</div>

A **production-grade AI system** for semiconductor wafer inspection. Built with **ResNet18** (Transfer Learning) to classify wafer defect patterns from map images. Integrated **Grad-CAM** for heatmap-based explainability вҖ” showing *exactly where* on the wafer the model detects anomalies. A **life-saving embedded system** built into a motorcycle helmet. The **MPU6050 IMU** continuously monitors 6-axis acceleration and gyroscope data. The moment a crash is detected (sudden impact threshold breach), the system autonomously fires a full emergency response chain вҖ” **no human intervention required.**

**Emergency Response Chain:**
- рҹ“Ў &nbsp;Reads **GPS coordinates** (lat/lng) from NEO-6M module
- рҹ“І &nbsp;Dispatches **SMS alert via GSM SIM800L** to emergency contacts
- рҹҡЁ &nbsp;Triggers onboard **buzzer + LED** for local warning
- вҡЎ &nbsp;**Sub-second end-to-end latency**
